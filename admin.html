Signature Variants
Event-themed signatures (school musicals, sports days, open days). Each variant inherits Nambour Christian College's contact details — only the logo and display name change. Staff opt in via their taskpane.
To push variants live: include them in the exported config JSON below as variants.json at the GitHub Pages root. Staff taskpanes auto-fetch this on launch.
Save Changes
How to ship these changes:
1. Upload your new image to WordPress at the Live logo URL above (use the Enable Media Replace plugin so the URL stays the same).
2. Paste the copied config JSON into taskpane.js — or into your SharePoint config list once that's built.
3. Re-publish the add-in. Staff pick up the new signature on next Outlook session.
